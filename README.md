# Lista de Tarefas (Kotlin)

Projeto do desafio técnico do estágio Android.

## Como executar
1. Clone o repositório:
    - git clone https://github.com/joaopfigo/kotlin-listadetarefas.git
2. Abra no Android Studio
3. Aguarde o Gradle Sync finalizar
4. Escolha um dispositivo (Emulador ou Celular com depuração USB)
5. Clique em Run (▶)

## Requisitos
- Android Studio (versão atual)
- SDK Android instalado pelo Android Studio
